## Installation and Setup Instructions

Installation:

`npm install`

To Run Test Suite:
- 9 test suites with 20 tests total

`npm test` || `yarn run test`

To Start Server:

`yarn start` || `npm start`

To Visit App:

`localhost:3000/`
